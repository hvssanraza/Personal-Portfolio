E‑Commerce Store

This C++ console application simulates a basic e-commerce platform featuring two user roles—customers and administrators—to manage shopping activities entirely via text-based navigation. Customers can register, log in, browse products, manage a shopping cart, complete purchases, and view order history. Administrators have elevated capabilities: viewing site-wide balance, managing inventory (adding products), processing orders (marking shipped), and updating admin credentials. Built with modular functions and data persistence through file-based I/O (users.txt, products.txt, orders.txt, etc.), the project includes input validation (e.g., verifying email format and password complexity), intuitive menu-driven navigation, and small UI enhancements like color-coded text and loading animations. Designed as an educational tool, it clearly demonstrates fundamentals of user authentication, file manipulation, role-based functionality, and console UI in C++.
